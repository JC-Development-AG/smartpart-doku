HATCHING - Schraffuren

Schraffuren werden von gefüllten 2D Polygonen und 2D Kurven verwendet.

DEFINE HATCHING name, id [, scale_dependent, angle, ref_x, ref_y, red, green, blue, alpha]

Parameter

name: Name der Schraffur
id: Allplannummer der Basisschraffur
scale_dependent: 0|1 = maßstabsunabhängige|maßstabsabhängige Darstellung
angle: Rotationswinkel der Schraffur(Standard: 0.0)
ref_x, ref_y: X- und Y-Koordinate des Referenzpunktes (Standard: (0.0, 0.0))
red, green, blue: (0..255) Farbwerte der Hintergrundfarbe (Standard: keine Farbe)
alpha: (0..100) Transparenzwert der Hintergrundfarbe

Mit HATCHING name wird die aktuelle Schraffur gesetzt.
