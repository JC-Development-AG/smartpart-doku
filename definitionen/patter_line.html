PATTER_LINE - Musterlinien

DEFINE PATTERN_LINE name, id [, width, height, align, type, ref_line, mirror, lock_corner, scale_dependent]

Parameter

name: Name der Musterlinie
id: Allplannummer der Basismusterlinie
width: Breite des Musters (Standard: 0.1)
height: Höhe des Musters (Standard: 0.1)
align: Ausrichtung 0|1|2 = links|mittig|rechts
type: Verschneidung 0|1|2|3 = keine|miter|join|seamless ???
ref_line: Darstellung der Referenzlinie 0|1 = aus|ein mirror: Muster ist 0|1 = nicht gespiegelt|gespiegelt
lock_corner: corner locking of the pattern 0|1 = not locked|locked ???
scale_dependent: 0|1 = maßstabsunabhängige|maßstabsabhängige Darstellung

Mit PATTERN_LINE name wird die aktuelle Musterlinie gesetzt.

Mit PATTERN_LINE -1 wird keine Musterlinie gesetzt.
