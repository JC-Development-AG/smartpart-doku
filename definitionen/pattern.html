PATTERN - Musterflächen

Muster werden von gefüllten 2D Polygonen und 2D Kurven verwendet.

DEFINE PATTERN name, id [, scale_dependent, angle, scale_x, scale_y, ref_x, ref_y, fitting_mode, red, green, blue, alpha]

Parameters

name: Name der Musters
id: Allplannummer des Basismusters
scale_dependent: 0|1 = maßstabsunabhängige|maßstabsabhängige Darstellung
angle: Rotationswinkel (Standard: 0.0)
scale_x, scale_y: Skalierung in X- und Y-Richtung (Standard: 1.0)
ref_x, ref_y: X- und Y-Koordinate des Referenzpunktes (Standard: (0.0, 0.0))
fitting_mode: 1|2|3 = outside fitting | fitting | inside fitting ???
red, green, blue: (0..255) Farbewerte der Hintergrundfarbe (Standard: keine Farbe)
alpha: (0..100) Transparenzwert der Hintergrundfarbe

Mit PATTERN name wird das aktuelle Muster gesetzt.
